Функционал приложения 8ЛР

GET-запросы на корневой путь '/' - ответом будет "Hello, world!"

GET-запросы на путь '/comments' - ответом будет JSON-массив объектов комментариев из базы данных в формате {name, text, date}

GET-запросы на путь '/comments/:id' - ответом будет конкретный комментарий из БД, совпадающий по ID

POST-запросы на путь '/comments' - добавление тела запроса в базу данных, в консоли обновлённый JSON-массив объектов комментариев из базы данных .
